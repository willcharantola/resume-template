# LaTeX Resume Template 

A professional, clean, and easy-to-use resume template built with LaTeX. **You don't need to be a programmer to use this!**

## üß© How to use (No coding required)

If you are not familiar with GitHub or coding, the easiest way to use this template is with **Overleaf** (a website that edits LaTeX documents online).

1.  **Download this template**: Click the green **Code** button at the top right of this page and select **Download ZIP**.
2.  **Create an account on Overleaf**: Go to [overleaf.com](https://www.overleaf.com) and sign up (it's free).
3.  **Upload the project**:
    - Click **New Project** -> **Upload Project**.
    - Select the ZIP file you just downloaded.
4.  **Choose your language**:
    - For an **English** resume, open the file `resumes/en/resume.tex`.
    - For a **Portuguese** resume, open the file `resumes/pt-br/curriculo.tex`.
5.  **Edit**: Just change the text inside the file.
    - *Example*: Change `\textbf{Your Name}` to `\textbf{John Doe}`.
    - Click **Recompile** to see your resume updated in PDF format!

## ‚ùì Frequently Asked Questions (FAQ)

<details>
<summary><strong>Which version should I use?</strong></summary>

- Use the **English version** (`/resumes/en/`) for international applications (US, Europe, Remote work).
- Use the **Portuguese version** (`/resumes/pt-br/`) for the Brazilian market.
</details>

<details>
<summary><strong>I don't know LaTeX. Can I still use this?</strong></summary>

**Yes!** As mentioned in the guide above, you just need to replace the placeholder text with your own information. You don't need to understand the complex code structure.
</details>

<details>
<summary><strong>Where can I get help with what to write?</strong></summary>

Check the `/prompts/` folder in this repository. There are some AI prompts included (for ChatGPT/Claude) that can help you write better bullet points for your experience.
</details>

<details>
<summary><strong>Should I put a photo on my resume?</strong></summary>

**No.** For most tech and professional jobs (especially international ones), photos are not recommended and can sometimes lead to automatic rejection by Applicant Tracking Systems (ATS).
</details>

<details>
<summary><strong>Can I convert this to Word?</strong></summary>

While possible, **it is not recommended**. One of the main benefits of LaTeX is that it creates a perfectly formatted PDF that looks professional on any device. Converting to Word often messes up the design.
</details>

<details>
<summary><strong>How do I generate the PDFs of my resumes?</strong></summary>

For each push (upload of code) to the repository, a workflow will automatically generate new PDFs with the latest changes. You can download them from the "Actions" tab in GitHub.
</details>
